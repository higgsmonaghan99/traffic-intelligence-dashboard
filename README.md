
# Traffic Intelligence Dashboard


## Features
- Interactive year range filter
- Multi-select facility filter
- Multi-select month filter
- KPI cards
- Traffic trend line chart
- Facility bar chart
- Facility share donut chart
- Monthly seasonality chart
- Premium dark glassmorphism UI

## Project structure
```bash
traffic-dashboard/
├── app.py
├── traffic_clean.csv
├── requirements.txt
└── assets/
    └── style.css
```

## Installation
```bash
pip install -r requirements.txt
```

## Run
```bash
python app.py
```

Then open the local Dash URL shown in the terminal.

## Recommended GitHub repo title
`traffic-intelligence-dashboard`

## Portfolio positioning
This project demonstrates:
- Data cleaning and type normalization
- BI dashboard design in Python Dash
- Interactive filtering and KPI reporting
- Time-series and categorical traffic analysis
